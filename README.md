# Sales Analytics Dashboard – Future Intern Task 1

## Project Overview

This project analyzes Superstore business sales data using Power BI to identify:

* Revenue trends
* Regional profitability
* Sales performance
* Business growth insights

## Tools Used

* Power BI
* Excel / CSV
* Data Visualization
* Business Analytics

## Dashboard Features

* KPI Cards
* Monthly Revenue & Profit Trend
* Regional Analysis
* Sales by Region
* Profit Insights

## Key Insights

* West region generated highest revenue
* Sales increased steadily over time
* Technology category showed strong performance
